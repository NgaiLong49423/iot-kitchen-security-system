# Project Demo & Presentation Resources

Thư mục này chứa các tài nguyên phục vụ cho việc chạy demo, cấu hình giao diện điều khiển (Dashboard) và minh họa hoạt động của hệ thống cho giảng viên.

## Nội dung thư mục đề xuất

Để tăng tính chuyên nghiệp, bạn có thể lưu trữ tại đây:

1.  **`Demo/Dashboard/`**:
    *   Lưu các hình ảnh chụp giao diện điều khiển trên điện thoại/máy tính (Arduino IoT Cloud Remote).
    *   File export cấu hình Dashboard hoặc mã cấu hình giao diện điều khiển (nếu có).
2.  **`Demo/SampleLogs/`**:
    *   File nhật ký hoạt động mẫu thu thập được từ thiết bị trong quá trình test thực tế.
    *   Ví dụ: File `.txt` ghi log các bản tin gửi lên Cloud, các cảnh báo thiết bị WiFi lạ lọt vào mạng, hoặc nhật ký lúc mất điện/có điện lại.
3.  **`Demo/Scripts/`**:
    *   Kịch bản chi tiết cho buổi thuyết trình demo các tính năng (đột nhập, vật nuôi, mất điện, WiFi lạ...).
    *   Link dẫn đến video Demo chính thức của nhóm (sau khi đã upload lên YouTube/Drive).

---

*Mục đích của thư mục này là giúp giảng viên hoặc người đánh giá dự án có thể nhanh chóng hình dung ra giao diện, kết quả chạy thực tế và kịch bản hoạt động của hệ thống mà không cần phải chạy lại phần cứng từ đầu.*
